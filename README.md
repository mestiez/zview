# zview

image viewer for x11

## Usage

`zview [filename or folder path]`

You can also drag the file into the window.

| Input     | Action |
|----------:|:-------|
| `LMB`     | Pan |
| `R`       | Rotate |
| `V`       | Flip (vertical) |
| `H`       | Flip (horizontal) |
| `Home`    | Reset camera |
| `.`       | Auto fit |
| `B`       | Toggle background |
| `Ctrl+V`  | View clipboard|
| `->`      | Next in dir |
| `<-`      | Previous in dir |
